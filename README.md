## 💻 Setup & Installation Guide

### Prerequisites
- Python 3.9 - 3.11
- Web Camera connected to PC
- C++ Compiler (Visual Studio C++ Build Tools on Windows; `build-essential` & `cmake` on Linux/macOS)

---

### Step-by-Step Execution

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/InDevTechnologies/facial-attendance-system.git
   cd facial-attendance-system
   ```

2. **Install Dependencies**:
   ```bash
   pip install -r setup_requirements.txt
   ```

3. **Run Application**:
   ```bash
   python app.py
   ```

4. Open `http://localhost:5000` in your browser, register student profiles, and log attendance using your webcam!
