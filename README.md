An automated facial recognition attendance management web system built with Python, Flask, OpenCV, and SQLAlchemy by **InDevTechnologies**.

---

## 🚀 Features

- **Live Camera Attendance**: Instant one-click face detection and attendance logging directly through browser webcams.
- **User Registration**: Simple interface for enrolling new faces into the encoding library.
- **Timestamped Dashboard**: Real-time attendance log visualization with date filtering.
- **Lightweight Database**: Powered by SQLite & SQLAlchemy ORM for zero-configuration local execution.

---

## 💻 Setup & Running

1. Clone the repository:
   ```bash
   git clone https://github.com/InDevTechnologies/facial-attendance-system.git
   cd facial-attendance-system
   ```

2. Install dependencies:
   ```bash
   pip install -r setup_requirements.txt
   ```

3. Run the application:
   ```bash
   python app.py
   ```

4. Open `http://localhost:5000` in your browser.
